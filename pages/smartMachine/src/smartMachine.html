  • Settings - Smart Ticketing Machine

    POS Settings Toggle →

    We've got three toggle buttons for the POS settings. Users can adjust them based on their preferences. Here's the information about the settings:

    • Print Report after ending trip: allows the bus owner to control whether the conductor can print the report after completing the trip. If this is turned on, only then will the conductor be able to print the report when the trip ends.
    • View Hisaab: configuration allows the bus owner to control whether the conductor is permitted to view the transaction details on the POS machine. If this option is turned on, only then will the conductor be able to see the transaction details on the POS machine.
    • Passengers: allows the bus owner to control whether the conductor is permitted to view the detail of the boarded and deboarded passengers on the POS machine.

    Password Management →

    Save Settings →

    Enables the user to save the changes made in the settings to the server. This ensures that the same adjustments will be applied to the chosen point-of-sale (POS) machine.
